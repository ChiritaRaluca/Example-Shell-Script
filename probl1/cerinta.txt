Să se scrie un shell script care primeşte un fişier file, o cifră c şi unul sau mai multe numere în linia de comandă. Dacă nu există minim 3 argumente se afişează mesaj de utilizare. Dacă c nu este cifră se afişează mesaj de eroare şi se încheie execuţia. Se afişează câte numere din listă au ultima cifră c. Se calculează suma tuturor numerelor şi se afişează suma şi câte cifre are această sumă. Dacă file reprezintă numele unui fişier obişnuit, se scriu în file suma numerelor şi câte cifre are suma. Dacă nu, se afişează un mesaj de eroare: file is not a regular file.
